# Mashov system Project

מערכת לניהול וצפייה בנתוני לימודים (מערכת שעות, ציונים ותזכורות) מבוססת Python. 
הפרויקט כולל צד לקוח (GUI) וצד שרת.

## מבנה הפרויקט

### קבצי קוד (Python)
* **main.py**: הקובץ הראשי של האפליקציה. מנהל את ממשק המשתמש (Tkinter), ה-Splash Screen ומעברי החלונות.
* **server.py**: צד השרת. מטפל בחיבורי לקוחות, אימות משתמשים (Login/Register) ושליפת נתונים מ-JSON.
* **src.py**: לוגיקה פנימית לניהול ציונים וחישוב ממוצעים.
* **getServersStrength.py**: סקריפט לבדיקת עומסים ותקשורת מול השרת (Multi-threading).
* **test.py**: בדיקות קטנות ליצירת סיסמאות אקראיות.

### נתונים (JSON / TXT)
* **schedule.json**: בסיס הנתונים של מערכת השעות לפי כיתות וימים.
* **reminder.json**: אחסון תזכורות ומטלות לשיעורים.
* **users.json**: רשימת המשתמשים הרשומים במערכת (נוצר ע"י השרת).
* **server_connection_logs.json**: תיעוד חיבורים וניסיונות כניסה לשרת.
* **privacy_policy.txt**: מסמך מדיניות הפרטיות של המערכת.
